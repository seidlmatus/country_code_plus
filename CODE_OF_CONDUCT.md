# Code of Conduct

## Our Commitment
We are committed to making participation in this project a respectful, harassment-free experience for everyone.

## Expected Behavior
- Be respectful and constructive.
- Focus on technical issues and collaboration.
- Assume good intent and give actionable feedback.

## Unacceptable Behavior
- Harassment, insults, or discriminatory language.
- Personal attacks or intimidation.
- Publishing private information without permission.

## Enforcement
Project maintainers may remove, edit, or reject contributions and comments that violate this Code of Conduct.

## Scope
This Code of Conduct applies in project spaces, including issues, pull requests, and discussions.
