# Security Policy

## Supported Versions

Security updates are provided for the latest published version on `main`.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report privately by emailing the maintainer:
- `matus.seidl@gmail.com`

Include:
- A clear description of the issue
- Reproduction steps or proof of concept
- Affected version(s)
- Potential impact

We will acknowledge receipt and provide status updates as triage progresses.
